# Learning compliant dynamical system from human demonstrations for stable force control in unknown environments

## Abstract
> This article presents a compliant dynamical system learning and control framework for robots to achieve stable force control in unknown environments. Specifically, a compliant dynamical system is learned from human demonstrations to transfer variable-impedance skills for delicate contact. The resulting controller maps force error to robot velocity while preserving stable robot-environment interaction, and the paper provides a Lyapunov-based stability analysis. The method is validated on both single-arm and dual-arm cooperative manipulation tasks.

## Why It Matters
- Represents the human-demonstration lineage for variable compliance, showing how force error and environment stiffness cues can be transferred into stable force control on UR5 single- and dual-arm tasks in unknown environments.
