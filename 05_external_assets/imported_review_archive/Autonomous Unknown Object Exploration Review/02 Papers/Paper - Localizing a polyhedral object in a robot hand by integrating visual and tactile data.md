# Localizing a polyhedral object in a robot hand by integrating visual and tactile data

## Abstract
> We present a novel technique for localizing a polyhedral object in a robot hand by integrating visual and tactile data. Localization is performed by matching a hybrid set of visual and tactile features with corresponding model features.

## Why It Matters
- An early direct formulation of camera-touch fusion for in-hand object localization; modern visuo-tactile pose estimators still solve the same ambiguity problem in richer sensing regimes.
- An early vision-touch localization formulation that already treats tactile data as the key to resolving occlusion and depth ambiguity during object localization in hand.
